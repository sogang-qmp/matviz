# 14.2 반응형 툴바

- `media/styles.css` `#top-bar`에 `container-type: inline-size; container-name: topbar` 선언.
- `@container topbar (max-width: 650px)` — `.bar-label` 안의 텍스트 숨김 (input 유지), 버튼·입력 축소.
- `@container topbar (min-width: 1200px)` — 버튼 여유 있게 확장.
- 브라우저가 container query 미지원일 경우 `@media (max-width: 650px)` 폴백.

# 14.3 단축키 도움말 모달

- `crystalEditorProvider.ts`: 모드바에 `?` 버튼, `#help-overlay` 모달 (배경 + 카드, 두 컬럼 그리드 — Rotation / Zoom / Navigation / Modes).
- `main.ts`: `?` 키 (Shift+/) · 버튼 · 닫기 `×` · 배경 클릭 · Esc 모두 처리.
- CSS: `.help-*` 컴포넌트 (`kbd` 스타일 포함).

# 14.4 상태 영속성 v1

- `renderer.getState()` 스키마 확장 (`schemaVersion: 1`) — polyhedra/boundary/cellDash 가시성, axisIndicatorSize, isoLevel, forceBonds, elementColorOverrides, elementRadiusOverrides, elementVisibility, bondOverrides 포함.
- `main.ts` `PersistedState`가 layoutMode + panelCollapsed/Width + stepAngle/stepZoom 추가. 관련 컨트롤 change/click 시 debouncedSave (300 ms).
- 탭 재오픈 시 렌더러와 UI 컨트롤 양쪽 복원. `schemaVersion !== 1` 은 폐기.

# 14.6 결합 스킵 UX

- `renderer.ts` `getBondSkipInfo()`, `setForceBonds()`, `forceBonds` 필드.
- Bonds 섹션 위에 `#bond-skip-hint` 블록 + `Compute anyway` 버튼.
- 노출 규칙: 원자 수 > 5000 && `forceBonds === false` 일 때만.
- 예상 시간: `atomCount × 0.03 ms` 상한 10 s.

# 14.7 AxisIndicator 추출

- 신규 `src/webview/axisIndicator.ts` — scene/camera/arrows/geometry/material/texture 자체 소유, `update`, `setSize`, `syncToMainCamera`, `render`, `dispose`.
